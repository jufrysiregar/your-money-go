## 📱 Tentang Aplikasi

**Your Money Go** adalah aplikasi manajemen keuangan pribadi yang membantu Anda mengelola pemasukan dan pengeluaran dengan lebih cerdas. Aplikasi ini dirancang untuk membantu Anda:

- 📊 Mencatat setiap transaksi keuangan
- 📈 Memantau arus kas harian, mingguan, bulanan, dan tahunan
- 💰 Mengontrol pengeluaran agar tidak boros
- 🎯 Merencanakan keuangan untuk masa depan

## ✨ Fitur Utama

### 🔐 Autentikasi Pengguna
- Registrasi akun baru dengan verifikasi hint keamanan
- Login dengan email dan password
- Lupa password dengan verifikasi hint
- Reset password melalui email dengan halaman custom

### 📊 Dashboard Keuangan
- **Total Pemasukan** - Ringkasan semua pemasukan
- **Total Pengeluaran** - Ringkasan semua pengeluaran
- **Sisa Saldo** - Perhitungan otomatis saldo terkini

### 📅 Manajemen Transaksi
- **Harian** - Catat transaksi per hari
- **Mingguan** - Lihat transaksi dalam seminggu
- **Bulanan** - Monitor pengeluaran bulanan
- **Tahunan** - Evaluasi keuangan tahunan

### 📈 Fitur Lainnya
- **Kategori Dinamis** - Pemasukan (Gaji, Bonus, THR) & Pengeluaran (Makanan, Transportasi, dll)
- **Grafik Ringkasan** - Visualisasi pengeluaran dengan chart
- **Tips Keuangan** - Saran cerdas mengelola uang
- **Hide/Show Password** - Keamanan tambahan
